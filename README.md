MERN Stack E-Commerce Website

A full stack E-commerce web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js) with modern UI, secure authentication, and complete shopping functionality including product management, cart, orders and payments.

-->FEATURES :

->User Features :

1. User registration and login (JWT authyentication)

2. Browse product by categories

3. Search and filter products

4. Product details with images and reviews

5. Add to cart and update quantity

6. Checkout with address and payment

7. Order history tracking

-> Admin Features :

1. Admin Dashboard

2. Add / Update / Delete products

3. Manage categories

4. Manage users

5. View and update order status

-> Payment Integration :

1. Razorpay / Stripe (can be configured)

2. Secure order confirmation

--> TECH STACK :

Frontend : React.js, HTML, CSS, JavaScript

Backend : Node.js, Express.js

Database : MongoDB

Authentication : JWT

State Management : Content API / Redux

Styling : CSS / Tailwind / Bootstrap

API Testing : Postman

--> INSTALLATION AND SETUP

1. Clone Repository

   git clone https://github.com/Nikhil-Prajapat-23/ecommerce-mern_stack

   cd ecommerce-mern_stack

2. Backend Setup

   cd backend

   npm install

   npm start

3. Frontend Setup

   cd frontend

   npm install

   npm start
